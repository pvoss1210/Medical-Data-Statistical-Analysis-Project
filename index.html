Two-Way ANOVA Procedure Overview

For two-way ANOVA, assuming the model assumptions are met, you first create an interaction plot to gauge if main effects are present for each factor and if there are interactions between the factors. If the lines corresponding to the levels of the factor not on the x-axis are not all the same line, this suggests that main effects are present for that factor. If the lines are not perfectly flat and horizontal, that suggests that main effects are present for the other factor. If the lines on the plot are parallel, this suggests that no interactions are present. To check for interactions further, you can use hypothesis testing with the null hypothesis being “(αβ)_ij = 0, ∀(i,j)” and the alternative hypothesis being “at least one (αβ)_ij != 0” with the p-value coming from the ANOVA table row that shows the predictors separated by a colon. If you are confident that interactions are present you should not test for main effects for each factor. If you are confident that interactions are not present, you can then test for main effects for each factor with the null hypothesis equal to “α_i = 0, ∀i” and the alternative hypothesis equal to “at least one α_i != 0” for the first factor, and the null alternative hypothesis equal to “β_j = 0, ∀j”and the alternative hypothesis equal to “at least one β_j != 0” for the second factor. The p-values for these tests come from the rows of the ANOVA table corresponding to each individual variable. If you tested for main effects and found factors were significant the next step is to conduct a Bonferroni and/or Tukey test, which can be used to test whether differences of levels within each factor are significant.

We do not need to test for main effects as interactions being present here means that the factors do not affect the dependent variable, BPSysAve, independently, so we cannot test whether the factors are significant individually as we do when we test for main effects.
